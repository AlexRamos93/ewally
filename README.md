# Desafio Ewally

## Getting Started

Clone or download the repository.

### Installing
```
$ npm install
```

### Running
```
$ npm start
```

### Testing
For running integration tests:

```
$ npm run test:integration
```

For running unit tests:

```
$ npm run test:unit
```

### Routes
/:boleto_number [GET]
```
http://localhost:8080/boleto/856900000584030100649158110347945609001374691358
```
